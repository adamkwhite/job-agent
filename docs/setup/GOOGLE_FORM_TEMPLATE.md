# Job Agent Profile Setup - Google Form Template

**Use this to create a Google Form in ~10 minutes**

---

## Setup Instructions

1. Go to https://forms.google.com
2. Click "+ Blank" to create new form
3. Title: "Job Agent Profile Setup"
4. Description: "Help us personalize your job search by answering these questions. This takes about 5-10 minutes."
5. Copy/paste questions below (in order)

## Form Settings (Important!)

After creating all questions:

1. **Click Settings (gear icon)**

2. **General tab:**
   - ✅ Collect email addresses
   - ✅ Limit to 1 response
   - Response receipts: "Always" (so they get a copy)

3. **Presentation tab:**
   - ✅ Show progress bar
   - Confirmation message: "Thanks! We'll set up your profile and send you a test digest within 1-2 days. Check your email for next steps."

4. **Responses tab:**
   - Click "Create spreadsheet" → New spreadsheet
   - Name it: "Job Agent Profile Responses"

---

## How to Use Responses

When someone submits the form:

1. Open the linked Google Sheet
2. Export the row as CSV or just copy the data
3. Use the responses to create `profiles/[name].json` following the template in `docs/development/ADDING_NEW_PROFILES.md`
4. Test with: `PYTHONPATH=$PWD job-agent-venv/bin/python src/send_profile_digest.py --profile [name] --dry-run`
5. Send them a test digest for feedback
